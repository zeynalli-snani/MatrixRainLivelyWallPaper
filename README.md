# 🌀 Interactive Matrix Wallpaper

An interactive, customizable Matrix rain live wallpaper designed specifically for **Lively Wallpaper**. It transforms the classic digital rain into a dynamic, physics-influenced canvas responsive to mouse movement, clicks, keyboard events.

![Wallpaper Preview](res/preview.gif)

## ✨ Features

* **Mouse Interaction:** Hovering over characters creates a bright white glowing effect.
* **Magenta Bursts:** Click anywhere on the screen or press any keyboard key to trigger localized digital explosions.
* **Fully Customizable:** Integrated directly with Lively Wallpaper's property listener for real-time adjustments.

## ⚙️ Customizable Properties

Through the Lively Wallpaper UI, you can dynamically customize:
* **Font Size:** Change the size of the digital rain characters.
* **Color:** Swap the traditional matrix green for any hex color.
* **Speed:** Control how quickly the characters cascade down the screen.
* **Trail Opacity:** Adjust the fade-out speed to create longer or shorter character trails.
* **Characters:** Input custom text, alphabets, or symbols to completely change the matrix stream.

## 🛠️ Built With

* **HTML5 Canvas** - High-performance 2D rendering.
* **Vanilla JavaScript** - Clean, library-free physics and animation logic.
* **CSS3** - Full-screen edge-to-edge styling.

## 🚀 How to Install in Lively Wallpaper

1. Download or clone this repository to your local machine.
2. Ensure your directory layout includes the following files:
   ```text
   ├── index.html
   ├── script.js
   ├── style.css
   └── res/
       └── preview.gif
       └── thumnail.png
   ```
3. Open **Lively Wallpaper**.
4. Click on **Add Wallpaper** (`+` icon).
5. Drag and drop the `index.html` file into Lively, or browse to select it.
6. Customize the properties directly inside the Lively UI to fit your setup!
